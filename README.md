# Expense_Tracker_Web-Tech 

## Overview  
Expense_Tracker_Web-Tech is a simple and responsive web application built using 
**HTML, CSS and vanilla JavaScript**, designed to help users track their daily income and expenses. 
Users can easily add expenses or income, view current balance, and maintain a history of their transactions — ideal for personal budgeting and financial awareness.

## Features 
- Add new transactions (income or expense) with description and amount.  
- See real-time updates of **balance**, **total income**, and **total expenses**.  
- View a list of all transactions.  
- Delete transactions as needed.  
- Responsive UI — works on desktop and mobile browsers.  
- No backend required — everything runs client-side (e.g. using local browser storage).
